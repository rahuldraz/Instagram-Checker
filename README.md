# Instagram-Checker
Checks accounts that dosent follow you back or you won't follow.

## Procedure
- Login to Instagram Web
- Copy the entire value of Cookie Header and paste in this script.

## Usage

````bash
./insta.py
````
#### Or

````bash
python3 insta.py
````
